# 新カードCore と Teppen本番COR 効果バランス分析レポート（詳細分類版）

## 概要

- **新カードCore**: 自作（180枚）
- **Teppen本番**: プロ制作（180枚）
- 同一カードが複数分類に含まれる場合あり（例：振り分けダメージ＋敵ヒーローダメージ）。

# 12. 「黒の破壊」「紫のバウンス」の代わりに、プロは何をしているか

自作では**黒の破壊**と**紫のバウンス**を多く作っている。その分、本番（プロ）は同じ色で別の効果に力を入れている。

## 12.1 黒：破壊を減らした代わりにやっていること

本番の黒は「コスト指定でガンガン破壊」より、**味方の死亡を前提にした設計**と**ライフ・墓地をリソースにしたプレイ**に振っている。

| プロがやっていること | 枚数・内容 |
|----------------------|------------|
| **＜リベンジ＞**（味方ユニットが死亡した時の反応） | **7枚以上**（ソドム、ゾンビ、ケルベロス、セス、T-002タイラント、ジェダ・ドーマ、悪魔との契約で付与）。「味方が死んだら自分が強くなる」が黒の軸。 |
| **ライフ犠牲**（プレイ時等にライフを払う） | **7枚**（ドスギルオス、ハンター、ウィリアム・バーキン、下僕の使役、心無き実験、サイコパワー等）。破壊ではなくライフでコストを払う。 |
| **成長・目覚め・解放・クエスト**（段階的強化） | ラドバルキン、レイダーキラー、ドスギルオス（クエスト）、VAVA、プロトタイラント、ウェスカー等。時間や条件で強くなる。 |
| **蘇生・墓地から戻す** | アーカム、ファントム、ベガ、ネルギガンテ、**強化蘇生**、**屍肉漁り**、集結する魂。破壊の対極で「死んだユニットを再利用」。 |
| **死亡時トリガー**（MP、ドロー、ダメージ、破壊等） | 多数。黒は「死んだとき」に発動する効果が豊富。 |
| **味方ユニット破壊を代償にした複合効果** | 自業自得、身勝手な捕食、狂気の殺戮。「破壊」は味方1体をコストにし、その見返りで除去・回復・ダメージ。 |
| **破壊そのもの** | 無条件は**抹殺**1枚（6コスト）。あとは攻撃力〇以下・ライフ15以下で条件変化など、**条件付き**が中心。 |

→ **まとめ**: 黒の「除去」は、**無条件破壊を少なくし**、代わりに**リベンジ・ライフ犠牲・蘇生・死亡時・成長**で「死と墓地」をテーマにしたデッキづくりをしている。

---

## 12.2 紫：バウンスを減らした代わりにやっていること

本番の紫は「EX・デッキに戻す」をあまり使わず、**停止**と**打ち消し**と**呼応**で妨害・連携を担っている。

| プロがやっていること | 枚数・内容 |
|----------------------|------------|
| **停止（halt）**（〇秒間行動不能） | **8枚以上**（ドスバギィ、アナカリス、アルケニー（死亡時）、ネルスキュラ、ウェブ・スパイダス、クリスタルハンター、ダークホールド、苦悩、モリガン）。紫の妨害の中心は「殺さずに止める」。 |
| **打ち消し（negate）**（アクションカードを無効化） | **5枚**（予期せぬ妨害、まやかしの情報、消された記憶、見え透いた嘘、隠された真意）。**本番では紫だけ**が持つ看板効果。バウンスではなく「出させない」。 |
| **デバフ**（攻撃力ダウン） | ゲリョス、ブーメル・クワンガー、ローズ、爪刈り、毒の散布、ウェブ・スパイダス。止めずに「弱くする」。 |
| **呼応（resonate）**（他カードとの連携で自分が強化） | **10枚前後**（プラズマ、ヘル＝ラスト、ヘル＝プライド、レイレイ、ヘル＝グリード、コーリン、いぶき、ジェスター、モリガン、ダンテ）。紫は「他と一緒にいると強くなる」カードが多い。 |
| **停止中の敵にダメージ** | ネルスキュラ、エイダ・ウォン、ウェブ・スパイダス（＋DOT付与）。停止とダメージのコンボ。 |
| **ヴェール・空戦**（対象にされない・飛行） | ディー、ブラッドゴイル、メフィスト、ゼノ・ジーヴァ、フックショット、魔人化。生存・回避寄り。 |
| **コントロール奪取** | 支配された頭脳（1枚）。バウンスの代わりに「奪う」。 |
| **対象ランダム変更** | 空蝉の術。相手のアクションの的をずらす。 |
| **バウンス** | **3枚のみ**（ゲリョス成長Lv2、こやし玉、魔力の封印）。EXに戻すは補助程度。 |

→ **まとめ**: 紫の「妨害」は、**バウンス（戻す）を少なくし**、代わりに**停止・打ち消し・デバフ・呼応**で「止める／無効化する／弱くする／連携で強くなる」に振っている。
