# 新カードCore 各色の特有効果一覧

`public/新カードCore - カードデータのスプレッドシート化.csv` をもとに、**色ごとの特有効果**（他色にほぼない・その色のアイデンティティになっているもの）をまとめたものです。デザインや新カード追加時の参照用です。

---

## 赤（Red）の特有効果

| 効果名 | 説明 | 主なカード例 |
|--------|------|----------------|
| **火種** | EXポケットに「火種」を加える。名前参照で他カード（火花の洗礼等）と連携。使用回数で効果が変化するカードあり。 | イグニス、ライガ、トラッカー、ガイド、紅蓮の幕開け、終末の開炉 |
| **EX・アクション連動** | 「EXポケットからカードを使ったとき」「アクションカードを使ったとき」で攻撃力+1・即時攻撃・ダメージ等。 | ライガ、アッシュ、ブレイカー、ガイド、爆奏のヴァルド |
| **攻撃準備時間短縮** | 味方ユニット1体の攻撃準備時間を〇秒短縮する。 | クーリエ、メッセンジャー |
| **効果ダメージで破壊したとき** | 自分の効果ダメージで敵を破壊したとき、自分や味方に+1/+1等のボーナス。 | スパーク、レグナ |
| **名前付きカード連携** | 「火種」「火花の洗礼」「残響点火」などカード名を指定した連携。 | ライガ、焔の洗礼式、残り焔の記憶、三連焔の起動 |
| **俊敏** | 出したターンに攻撃可能（文言は「俊敏」）。 | ブレイカー、ハンター |
| **アクションのダメージ増加** | 自身のアクションカードのダメージを+1する。 | スポッター |

※赤は「ダメージ」「攻撃力バフ」も多いが、他色にもあるため、上記は**赤で特に目立つ**効果として列挙。

## 緑（Green）の特有効果

| 効果名 | 説明 | 主なカード例 |
|--------|------|----------------|
| **棘** | 攻撃してきた敵に〇ダメージを与える。棘：1〜5 の付与・棘を+1〜+3。棘を持つ味方にバフするカードあり。 | タイタン、ソウヤー、ワッチ、ヘラルド、荊棘の交響、棘の鎧、棘魂のロード |
| **攻撃した敵に〇ダメージ** | 棘と同系統。「このユニットに攻撃した敵に〇ダメージ」「受けたダメージ+〇」など。 | 硬芽のバインダー、棘冠のドラゴン、リベラー、棘返しの契り、荊王の刻印 |
| **＜シールド＞** | ダメージを防ぐ。1〜3まで。棘とセットで付与されることが多い。 | エルダー、ワッチ、テイラー、硬芽のバインダー、棘魂のロード、古樹の皮 |
| **硬化** | 受ける戦闘ダメージを〇軽減（〇秒間）。棘デッキと組み合わせが多い。 | ミキサー、棘壁の刻印、硬化の輪、森羅の結界、大樹の号令 |
| **不屈** | HPが0になるダメージを1回だけHP1で耐える。 | キーパー、不屈の祝福、棘血の盟約 |
| **護符** | 次に受ける戦闘ダメージを-〇（1回）、または効果ダメージを無効化（1回）。 | ポリッシャー、棘の返礼 |
| **再生** | 〇秒ごとにHPを1回復（〇秒間）。 | 棘返しの契り |
| **前借り** | 味方ヒーローがMPを〇だけ前借りする。その分、一定時間MPの溜まる速度が下がる（または0になる）。 | シャーマン、森契のペイヤー、借時の対価 |
| **MPブースト** | 味方ヒーローにMPブースト〇（秒数制限あり/なし）を付与。 | メンター、森の奔流、長命の加護、緑の契約、聖樹の抱擁 |
| **〇秒後／〇秒ごと** | 時間経過でバフ・回復・シールド付与。 | ガーディアン、ディフェンダー、エルダー、蘇生の息吹、硬芽のバインダー |

※緑は「回復」「HPバフ」も多いが、**棘・硬化・不屈・前借り・MPブースト**が緑特有のキーワードとしてまとまる。

## 紫（Purple）の特有効果

| 効果名 | 説明 | 主なカード例 |
|--------|------|----------------|
| **手札公開** | 相手の手札を1枚（以上）公開させる。公開されていることを条件にした効果とセット。 | ミスト、ウォッチャー、手札混線 |
| **公開札のコスト操作** | 公開されている相手の手札のコストを+1〜+2（1枚 or すべて）。時間指定（〇秒間／このバトル中）あり。 | ジャマー、プレッシャー、バインド、ウェイト、ロック |
| **停止** | 敵ユニットを〇秒間行動不能にする。1体／ランダム1体／正面／すべて。5秒 or 13秒。 | スタナー、ペトリファー、スタッガー、永続の縛、正面封鎖、全員静止 |
| **毒** | 敵に毒：1〜3 を付与。毒状態の敵をEX・デッキに戻す、毒即時爆発、毒増幅など毒専用カードあり。 | ポイズナー、ベノマー、毒牙の一刺し、毒の拡散、毒即時爆発、毒増幅 |
| **バウンス（戻す）** | 敵ユニットをEXポケット／デッキの一番上・一番下・ランダムな位置に戻す。コスト条件・停止状態・毒状態と組み合わせあり。 | リバウンダー、リクレイマー、リターナー、ボトム送り、バウンス、シャッフラー、デッキ上送り、沈殿ドロー、虚空の重還 |
| **戻したカードのMP変更** | EXに戻すときに、そのカードのMPを-1〜-3する（最低1）。 | リクレイマー、正面強制還却、虚空の重還 |
| **EXポケット操作（相手）** | 相手のEXのカードを1枚見て、一番上か一番下に置き直す。 | オーダー |
| **手札撹乱** | 相手の手札を墓地に捨てさせる、またはデッキのランダムな位置に戻す。 | 手札撹乱、手札混線、スパイ |
| **停止＋ダメージ／停止＋戻す** | 停止状態の敵にダメージを与える、または停止状態の敵を戻す。 | フィニッシャー、リクレイマー、停止狩り、デッキ送還、二重処分 |

※紫は「毒」「停止」「手札公開・コスト操作」「バウンス」がセットで、**情報妨害と時間差除去**が紫の軸。

## 黒（Black）の特有効果

| 効果名 | 説明 | 主なカード例 |
|--------|------|----------------|
| **墓地枚数条件** | 「自分の墓地が〇枚以上なら〜」。閾値で効果が変わる（3枚/4枚/5枚/6枚/7枚/8枚/10枚など）。バフ・回復・破壊・ダメージに使用。 | ハーヴェスター、グラッシャー、シード、ヒーラー、墓地破壊、収穫の恵み、終焉の殲滅 |
| **【代償】** | 効果のコストとして「手札を捨てる」「味方ヒーローのライフを減らす」「味方ユニット1体を破壊する」を明示。 | ディスカード、サクリファイス、代償の一撃、血の代償、二重破壊、回復の代償 |
| **コスト指定破壊** | コスト〇以下の敵ユニット1体（または2体まで）を破壊する。1〜7まで段階あり。正面限定・ランダムあり。 | クラッシャー、デストロイ、エリミネーター、オブリテイター、二体破壊、終焉の殲滅 |
| **破壊したカードを除外** | 破壊したユニットを墓地ではなく除外する。死亡時能力を発動させない意図。 | 大半の黒の破壊・ダメージカード |
| **死亡時** | 自分の死亡時に、墓地枚数条件付きで味方にバフ等。 | ヘイラー、レガシー |
| **コスト指定ダメージ＋除外** | コスト〇以下の敵1体にダメージを与え、破壊した場合除外。 | スラッシャー、リーパー、四重撃、六重撃等 |
| **墓地枚数でダメージ量変化** | 墓地が〇枚以上なら〇ダメージ、〇枚以上なら代わりに〇ダメージ。 | ショット、ダブルタップ、リーパー |

※黒は「破壊」「除外」「墓地」「代償」が一体で、**リソース（手札・ライフ・味方ユニット・墓地）を消費して除去・バフ**する設計。

| 色 | 特有効果のキーワード（短く） |
|----|------------------------------|
| **赤** | 火種、EX・アクション連動、攻撃準備時間短縮、効果ダメージ破壊ボーナス、名前連携、俊敏 |
| **緑** | 棘、攻撃した敵にダメージ、＜シールド＞、硬化、不屈、護符、再生、前借り、MPブースト、時間経過バフ |
| **紫** | 手札公開、、停止、毒、バウンス（EXに戻す）、EX操作、手札破壊 |
| **黒** | 墓地枚数条件、【代償】、コスト指定破壊、除外、死亡時、墓地でダメージ量変化 |
