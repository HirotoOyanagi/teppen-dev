## 赤カード「弱いだろ」候補（暫定）

評価基準：**コスト / 攻撃力 / HP / 効果**を総合して、同コスト帯の赤カード（特に除去性能・テンポ・即効性）と比較して見劣りするもの。

※CSV修正で数値/効果が変わったカードは、カード名の後ろに**（修正済）**を付けています。

### ユニット

- **COR_019｜ダスター（修正済）｜2MP｜攻2 / HP4**
  - **効果**：登場時：正面の敵ユニット1体に「そのユニットが受ける効果ダメージを+3する」を付与する。
  - **弱い理由（総合）**：効果が**即時の除去/打点に直結しにくい**。赤の同コスト帯は直接ダメージが強く、採用するとテンポ負けしやすい。

- **COR_021｜シールド｜3MP｜攻2 / HP6**
  - **効果**：登場時：このユニットはアクションカードのダメージを受けない。
  - **弱い理由（総合）**：3MPで攻2は圧が弱い。耐性が「アクションのダメージ」限定で、刺さらない相手だと**コストに対してただ硬いだけ**になりやすい。

- **COR_024｜アーマー｜3MP｜攻1 / HP7**
  - **効果**：登場時：味方ユニット1体に+2攻撃力を付与する。その後、そのユニットのHPが最大値なら、代わりに恒久的な+2攻撃力を付与する。
  - **弱い理由（総合）**：本体攻1でテンポが出にくい。バフ先・盤面状況に依存しやすく、恒久条件も不安定で**3MPの即効性に欠ける**。

- **COR_020｜プロード（修正済）｜3MP｜攻2 / HP5**
  - **効果**：登場時：味方ユニット1体に「攻撃時：正面のユニットに自身の攻撃力分のダメージ与える」を付与する。
  - **弱い理由（総合）**：付与自体は強い可能性があるが、**対象依存・正面依存**でブレが大きい。3MPでこの不安定さは採用理由が作りづらい。

### アクション

- **COR_041｜側面強襲（修正済）｜2MP**
  - **効果**：ランダムな敵ユニットに4ダメージを振り分ける。このアクションの効果で敵ユニットが破壊された場合、敵リーダーに1ダメージ。
  - **弱い理由（総合）**：合計4点は強いが、**ランダム**で必要な場所に当たりにくい。確実性が要る局面では役割を持ちにくい。

- **COR_044｜完全制圧（修正済）｜3MP**
  - **効果**：全ての敵ユニットに2ダメージ。全ての敵ユニットは反撃することができなくなる。
  - **弱い理由（総合）**：全体2点は偉いが、3MPとしては**取り切れない場面が多い**。反撃不可も盤面依存で、純粋な除去としては上振れが必要。

- **COR_043｜戦慄の圧迫（修正済）｜2MP**
  - **効果**：敵ユニット1体に2ダメージ。13秒間にそのユニットが受けるダメージを+2する。
  - **弱い理由（総合）**：2MP枠としては「即時に倒す」力が弱く、**追撃前提**になりやすい。赤の火力カードと比べて枠競争で負けやすい。

### 同系統効果で「下位互換っぽい」候補

- **COR_004｜バーン｜3MP｜攻3 / HP4**
  - **効果**：攻撃時：正面のユニットに2ダメージ。
  - **比較先**：**COR_008｜ラッシュ｜3MP｜攻3 / HP4**（攻撃時：正面の敵ユニットに3ダメージ）
  - **下位寄り理由**：同コスト同スタッツで効果だけ弱いので、**ほぼ下位互換**。

- **COR_042｜拡散射撃｜2MP**
  - **効果**：敵ユニットに3ダメージを振り分ける。
  - **比較先**：**COR_041｜側面強襲（2MP）**（ランダムとはいえ合計4点＋破壊で顔1点）
  - **下位寄り理由**：役割が近い「小分け除去」枠で、総量が見劣りしやすい（※3点を狙って配れる価値はある）。

- **COR_038｜三連焔の起動｜3MP**
  - **効果**：「火種」が3枚以上使用していた場合、敵ユニット1体に6ダメージ。その後、自分のEXポケットの「火種」を2枚まで墓地に送り、その枚数だけ敵ヒーローに1ダメージ。
  - **比較先**：**COR_034｜爆心撃｜3MP**（敵ユニットに6ダメージ。破壊で正面味方に+3/+3）
  - **下位寄り理由**：「3MPで6点除去」という役割で見ると、三連焔は**条件（火種3回）**が重く、追加価値もEX火種消費でコストがかかるため、安定性で下位寄りになりやすい。
