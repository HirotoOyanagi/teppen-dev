## 赤単デッキ案（30枚構成）

レジェンドは重複なし、その他は最大3枚までの条件で、別コンセプトの赤単デッキを3種類作成しました。

---

## デッキ1：火種コンボバーン

**コンセプト**:  
「火種」を軸に盤面除去とリーダー火力を両立するミッドレンジ寄りコンボデッキ。小型ユニットで「火種」を確保しつつ、`ツインバレット`、`ブレイカー`、`紅蓮の幕開け`、`焔の収束点`、`終末の開炉`で一気にゲームを決めに行きます。

**デッキリスト（30枚）**

- ユニット（18枚）
  - `イグニス` ×3  
    - 効果：登場時：自身のEXポケットに「火種」を1枚加える。自身がアクションカードを使ったとき、このユニットの攻撃力を＋1。
  - `トラッカー` ×3  
    - 効果：登場時：デッキから「火種」1枚を探してEXポケットに加える。
  - `メッセンジャー` ×3  
    - 効果：登場時：デッキから「火種」1枚を探してEXポケットに加え、デッキをシャッフルする。その後、味方ユニット1体の攻撃準備時間を2秒短縮する。
  - `ガイド` ×2  
    - 効果：登場時：「火種」1枚を探して手札に加える。自分がアクションカードを使ったとき、そのカードが「火種」ならこのユニット攻撃力を+1する。
  - `スキャッター` ×2  
    - 効果：登場時：敵ユニット1体に1ダメージ。対象のHPが3以下なら、代わりに3ダメージ。
  - `ツインバレット` ×2  
    - 効果：登場時：敵ユニット1体に4ダメージ。「火種」を3枚以上使っていた場合、代わりに7ダメージを与え、敵ヒーローに2ダメージ与える。
  - `ブレイカー` ×2  
    - 効果：「火種」を3枚以上使用していた場合、攻撃時:5ダメージを与えるを得る。俊敏
  - `爆奏のヴァルド` ×1（レジェンド）  
    - 効果：登場時：自身の使用したアクションカードの数、敵ユニットに1ダメージを割り振る。自身がEXポケットからカードを使うたび、敵リーダーに2ダメージ。
  - `煉獄の覇者 カイゼル` ×1（レジェンド）  
    - 効果：登場時：敵リーダーに自信のユニットが受けるダメージをすべて+1するを付与する。ランダムな敵ユニット一体に7ダメージを与える。

- アクション（12枚）
  - `火種` ×3  
    - 効果：敵ユニット1体に2ダメージ。
  - `紅蓮の幕開け` ×2  
    - 効果：自身のEXポケットに「火種」を2枚加える。敵ユニット全体に5ダメージ。敵リーダーに３ダメージ
  - `三連焔の起動` ×2  
    - 効果：「火種」が3枚以上使用していた場合、敵ユニット1体に6ダメージ。その後、自分のEXポケットの「火種」を2枚まで墓地に送り、その枚数だけ敵ヒーローに1ダメージ。
  - `焔の収束点` ×2  
    - 効果：敵ユニットに「火種」を使用した回数×2ダメージを振り分ける。
  - `終末の開炉` ×1（レジェンド）  
    - 効果：「火種」を使用した回数だけすべての敵ユニットと敵リーダーに1ダメージを与える。
  - `灼熱の一閃` ×2  
    - 効果：敵ユニット1体に4ダメージ。アクティブレスポンス中に使った場合、代わりに6ダメージ。このカードの効果で敵ユニットを破壊した時、敵ヒーローに1ダメージ。

**採用理由・役割**

- **`イグニス` / `ガイド`**:  
  アクション使用や「火種」そのものを手札に加える役。序盤からプレイしておくことで、中盤以降のアクション連打と`爆奏のヴァルド`のダメージ源を確保します。

- **`トラッカー` / `メッセンジャー` / `火種`**:  
  デッキ・EXポケットから安定して「火種」を確保するパッケージ。`三連焔の起動`、`焔の収束点`、`終末の開炉`、`ブレイカー`、`ツインバレット`など「火種」依存カード群の条件達成を早めます。

- **`スキャッター`**:  
  1MPで小型を触れる優秀なテンポ除去。序盤の盤面処理＋「火種」で削った中型にも届きやすく、相手のテンポを崩します。

- **`ツインバレット` / `ブレイカー`**:  
  火種3回以上使用を満たした中盤以降の主力。`ツインバレット`はユニット除去＋リーダー2点、`ブレイカー`は俊敏かつ攻撃時5点でゲームを一気に閉じに行けます。

- **`爆奏のヴァルド`**:  
  これまで使ったアクションの数だけダメージを割り振れるフィニッシャー。EXポケットからカードを使うたび敵リーダー2ダメージなので、「火種」や`残響点火（残り焔の記憶）`との相性も抜群です。

- **`煉獄の覇者 カイゼル`**:  
  敵ユニット7ダメージ＋敵リーダーへの受けるダメージ+1付与で、以降のバーンカード全体を実質強化するレジェンドフィニッシャー。

- **`紅蓮の幕開け`**:  
  「火種」2枚供給＋全体5ダメージ＋リーダー3ダメージ。火種シナジーの加速と盤面一掃、リーサル圏内の押し込みを一枚でこなす超重要カード。

- **`三連焔の起動` / `焔の収束点` / `終末の開炉`**:  
  火種使用回数が増えるほど伸びるバーンパッケージ。序盤からこつこつ「火種」を使っておくことで、これらのカードで大きなリターンを得られます。

- **`灼熱の一閃`**:  
  単体除去＋条件付きで6ダメージ。ユニット処理からフィニッシュまで柔軟に使える、火力の厚みを支える1枚。

## デッキ2：効果ダメージ強化＆盤面制圧

**コンセプト**:  
`スポッター`、`レグナ`、`ダスター`、各種バーンアクションで効果ダメージを増幅しながら、中〜大型ユニットを安全に処理していくコントロール寄り赤デッキ。盤面を完全に支配しつつ、`レグナ`のバフで自軍ユニットを育てて勝ちます。

**デッキリスト（30枚）**

- ユニット（19枚）
  - `スキャッター` ×3  
    - 効果：登場時：敵ユニット1体に1ダメージ。対象のHPが3以下なら、代わりに3ダメージ。
  - `スポッター` ×3  
    - 効果：自身のアクションカードのダメージを+1する。
  - `ダスター` ×3  
    - 効果：登場時：味方ユニット1体に「そのユニットが受ける効果ダメージを+3する」を付与する。
  - `バーン` ×2  
    - 効果：攻撃時：正面のユニットに2ダメージ。
  - `マルク` ×2  
    - 効果：登場時：正面以外の敵ユニットに5ダメージ。
  - `レグナ` ×2  
    - 効果：登場時：敵ユニット1体に3ダメージ。自身の効果ダメージで敵ユニットが破壊されるたび、ランダムな味方ユニット1体に+2/+2を付与する。
  - `シールド` ×2  
    - 効果：登場時：このユニットはアクションカードのダメージを受けない。
  - `黎明の守護者レイシア` ×1（レジェンド）  
    - 効果：登場時：味方ユニットすべてに、それぞれの正面の敵ユニットの攻撃力と同じ値の攻撃力強化を付与する
  - `爆奏のヴァルド` ×1（レジェンド）  
    - 効果：登場時：自身の使用したアクションカードの数、敵ユニットに1ダメージを割り振る。自身がEXポケットからカードを使うたび、敵リーダーに2ダメージ。

- アクション（11枚）
  - `火種` ×2  
    - 効果：敵ユニット1体に2ダメージ。
  - `灼熱の一閃` ×2  
    - 効果：敵ユニット1体に4ダメージ。アクティブレスポンス中に使った場合、代わりに6ダメージ。このカードの効果で敵ユニットを破壊した時、敵ヒーローに1ダメージ。
  - `鉄壁貫通撃` ×3  
    - 効果：敵ユニット1体に5ダメージを与える。
  - `爆心撃` ×2  
    - 効果：敵ユニットに6ダメージ。破壊した場合、その正面の味方ユニット1体に+3/+3を付与。
  - `戦慄の圧迫` ×2  
    - 効果：敵ユニット1体に2ダメージ。13秒間にそのユニットに戦闘ダメージ＋２を付与

**採用理由・役割**

- **`スポッター` / `ダスター`**:  
  効果ダメージの基礎火力を底上げする要。`ダスター`の「受ける効果ダメージ+3」付与と`スポッター`の「アクションダメージ+1」が乗ることで、`火種`や`鉄壁貫通撃`などの軽いカードで高耐久ユニットまで処理できます。

- **`スキャッター` / `バーン` / `マルク`**:  
  1〜5コスト帯の即時除去要員。`スキャッター`は小型処理、`バーン`は戦闘ダメージ2点内蔵、`マルク`は正面以外5ダメージで並びを崩せます。

- **`レグナ`**:  
  効果ダメージで敵ユニットを破壊するたび味方に+2/+2を振りまくキーカード。このデッキの豊富なバーンと組み合わせると、盤面処理しながらサイズの大きなフィニッシャーを自然と作り上げられます。

- **`シールド`**:  
  アクションダメージを受けないタンク役。相手の火力デッキに対し前線を張りつつ、背後に`スポッター`や`レグナ`など重要ユニットを安全に並べやすくします。

- **`黎明の守護者レイシア`**:  
  敵ユニットの攻撃力を参照した一斉バフで一気に打点を増やすフィニッシャー。相手が高攻撃力ユニットを横に並べるほど、こちらのリーサル圏が近づきます。

- **`爆奏のヴァルド`**:  
  アクションの枚数が自然と多い構成なので採用。中〜終盤に出すことで、盤面をほぼ一掃しつつ、以降のEXポケットカード使用で継続的に敵リーダーへダメージを飛ばせます。

- **`火種` / `灼熱の一閃` / `鉄壁貫通撃` / `爆心撃` / `戦慄の圧迫`**:  
  このデッキの心臓部。`戦慄の圧迫`で戦闘ダメージを増幅してから殴る、`爆心撃`で処理＋味方バフを同時に行う、`灼熱の一閃`や`鉄壁貫通撃`で単体を確実に落とすなど、状況に応じた柔軟な除去ラインを取れるように枚数を調整しています。

## デッキ3：速攻アグロラッシュ

**コンセプト**:  
低〜中コストユニットをテンポ良く並べ、`コマンダー`や`アーマー`、各種バフアクションで一気にリーダーライフを削り切るアグロ寄りの赤単。細かい除去よりも、攻撃準備時間短縮と連撃・無敵突破で「通る打点」を最大化します。

**デッキリスト（30枚）**

- ユニット（21枚）
  - `アッシュ` ×3  
    - 効果：EXポケットからアクションカードを使ったとき、このユニットは即時攻撃を行う。
  - `クーリエ` ×3  
    - 効果：登場時：味方ユニット1体の攻撃準備時間を5秒短縮する。
  - `トラッカー` ×2  
    - 効果：登場時：デッキから「火種」1枚を探してEXポケットに加える。
  - `ラッシュ` ×3  
    - 効果：攻撃時：正面の敵ユニットに3ダメージ
  - `コマンダー` ×3  
    - 効果：登場時：味方ユニットすべてに+1攻撃力を付与する。さらに、現在のMPが5以上なら、ランダムな味方ユニット1体にさらに+1攻撃力を付与する。
  - `アーマー` ×2  
    - 効果：登場時：味方ユニット1体に+2攻撃力を付与する。その後、そのユニットのHPが最大値なら、代わりに恒久的な+2攻撃力を付与する。
  - `ハンター` ×3  
    - 効果：登場時：俊敏。攻撃時：このユニットの攻撃力を+1する。
  - `ブレード` ×2  
    - 効果：相手がアクションカードを使用した時、即時攻撃を行う
  - `黎明の守護者レイシア` ×1（レジェンド）  
    - 効果：登場時：味方ユニットすべてに、それぞれの正面の敵ユニットの攻撃力と同じ値の攻撃力強化を付与する

- アクション（9枚）
  - `火種` ×2  
    - 効果：敵ユニット1体に2ダメージ。
  - `焔の洗式` ×3  
    - 効果：自身のユニット1体に+3攻撃力を付与する。そのユニットがこのターン敵ユニットを撃破したとき、自身のEXポケットに「火種」を1枚加える。
  - `無敵突破` ×2  
    - 効果：自身のユニット1体は敵ユニットにブロックされないを得る（一回のみ）。さらに+2攻撃力を得る。
  - `斬り込み攻撃` ×2  
    - 効果：味方ユニット1体に、その正面に敵ユニットがいるなら+5攻撃力を付与する。いないなら、代わりに+1攻撃力と「連撃」を付与する。

**採用理由・役割**

- **`アッシュ` / `クーリエ`**:  
  攻撃準備時間を短縮しつつ、EXポケットアクションとのシナジーで即時攻撃を狙える序盤のキーパーツ。特に`アッシュ`はEXポケットからのアクション使用で即時攻撃を行うため、`火種`や`焔の洗式`と絡めて予想外の打点を叩き込めます。

- **`トラッカー` / `火種` / `焔の洗式`**:  
  打点補助と除去を兼ねた軽量バーンパッケージ。`焔の洗式`で強化したユニットが敵を撃破すればEXポケットに「火種」が補充されるため、アクション連打によるテンポ維持がしやすくなります。

- **`ラッシュ` / `ハンター`**:  
  攻撃時効果や俊敏で即時打点を出せる主力アタッカー。`ラッシュ`は正面3ダメージ、`ハンター`は攻撃するたび自己強化と、どちらも放置されると相手にとって大きなプレッシャーになります。

- **`コマンダー` / `アーマー`**:  
  全体・単体バフ要員。`コマンダー`は横並びを一気に強化し、MP5以上ならさらに+1攻撃力でフィニッシュを後押し。`アーマー`はフルHPユニットに恒久+2攻撃力を与えられるため、早い段階で付与できればそのユニットだけでゲームを決めることも可能です。

- **`ブレード`**:  
  相手のアクションに即時反応して攻撃できるカウンターアタッカー。除去やバフアクションに対してテンポを取り返しやすく、アグロが苦手とする「相手の受け」に圧力をかけます。

- **`黎明の守護者レイシア`**:  
  終盤の押し込み用フィニッシャー。相手の攻撃力が高いほどこちらの打点も一気に跳ね上がるため、アグロミラーや中〜大型ユニットが多いデッキに対して特に刺さります。

- **`無敵突破` / `斬り込み攻撃`**:  
  ブロッカーを無視してリーダーにダメージを通すためのフィニッシュパーツ。`無敵突破`はブロック不可＋攻撃力+2、`斬り込み攻撃`は状況に応じて+5か連撃＋小強化を選べるため、どちらも終盤リーサル計算を大きく変えるカードです。

---

以上が、赤単の別コンセプト3デッキ案です。  
必要であれば、この中から気に入ったコンセプトを一つ選んでもらい、より細かいマリガン方針やプレイング指針もまとめられます。
